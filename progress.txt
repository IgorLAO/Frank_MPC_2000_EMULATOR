## Codebase Patterns
- Rust 2024 edition with eframe 0.31, egui 0.31, rodio 0.20, cpal 0.15
- Build requires: pkg-config, libasound2-dev, build-essential (gcc)
- Use `cargo check` for typecheck, `cargo build` to compile
- eframe::run_native takes a closure returning `Ok(Box::new(App))`
- egui::CentralPanel::default().show() for main UI layout
